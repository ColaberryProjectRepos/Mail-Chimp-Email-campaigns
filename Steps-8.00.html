Creating Package in SSIS 8

- Open Microsoft Visual Studio 

- Click File --->  Click New --->  Click Project

- In New Wizard ---> Choose Integration Services ---> Choose Integration Services Project ---> In Name give MailChimp-Marketing Project ---> OK

- Rename the Package ---> Right Click on Package.dtsx in the Solution Explorer ---> Rename ---> Rename it to MailChimp_Marketing.

- Drag and Drop Data Flow Task into Control Flow Task

- Double Click to Data Flow Task
